# Listbox

<Description of={Listbox} />

## Usage

```tsx
import { Listbox } from '@divriots/starter-tailwindcss/listbox';

<Listbox
  items={[
    { name: 'Wade Cooper' },
    { name: 'Arlene McCoy' },
    { name: 'Devon Webb' },
    { name: 'Tom Cook' },
    { name: 'Tanya Fox' },
    { name: 'Hellen Schmidt' },
  ]}
  onSelected={({ name }) => window.alert(`You selected ${name}`)}
/>;
```
